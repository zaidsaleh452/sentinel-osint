// PROJECT SUMMARY (LINKEDIN-READY)

Built Sentinel OSINT, a self-contained, single-file Threat Intelligence Dashboard, to strengthen my hands-on cybersecurity and front-end development skills. The project integrates a live REST API (SANS Internet Storm Center InfoCon feed) using the native JavaScript Fetch API, with asynchronous async/await logic and graceful error handling for offline/CORS-restricted environments. It parses and renders JSON data dynamically, manipulates the DOM in real time, and includes an interactive IP/domain/hash analyzer that simulates threat-scoring logic. The interface is a fully responsive UI built with Tailwind CSS, featuring a custom dark "command center" design system, CSS animations, and accessibility considerations (reduced-motion support, semantic markup). No frameworks, build tools, or backend required — just HTML, CSS, and vanilla JavaScript in a single file.

JavaScript (ES6+) Fetch API / REST JSON Parsing Async/Await DOM Manipulation Tailwind CSS Responsive Design Error Handling
