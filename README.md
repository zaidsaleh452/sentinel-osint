# SENTINEL_OSINT: Threat Intelligence Dashboard

An interactive, single-file OSINT (Open Source Intelligence) Command Center designed for security analysts to monitor global internet threat levels in real-time. Built as a practical portfolio project transitioning into Year 2 Cyber Security at Leeds Beckett University.

## Live Demo
[CLICK HERE TO VIEW THE LIVE DASHBOARD](https://zaidsaleh452.github.io/sentinel-osint/)

## Features
* **Live SANS InfoCon Integration:** Fetches real-world internet threat levels (Green/Yellow/Red) directly from the SANS Internet Storm Center API.
* **Simulated Threat Analyzer:** Validates and analyzes IP addresses, domains, and file hashes using deterministic regex routing to simulate SOC workflows.
* **Cyber Command Center UI:** Designed with a sleek, responsive, dark-mode terminal aesthetic using Tailwind CSS.

## Core Skills Demonstrated
* **API Handling & Asynchronous JS:** Implementing `fetch()` and managing CORS/local fallback logic.
* **Data Parsing:** Processing live JSON data streams into user-facing security metrics.
* **Front-End Design:** Crafting clean, responsive UI layouts using utility-first CSS frameworks.
